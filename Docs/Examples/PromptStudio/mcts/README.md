# Prompt Studio MCTS Examples

This folder contains release-ready example payloads referenced by the Prompt Studio MCTS guides and API docs.

## Files

- `create_optimization_mcts.json`: request body for `POST /api/v1/prompt-studio/optimizations/create`
- `websocket_optimization_iteration_event.json`: example `optimization_iteration` WebSocket event
- `optimization_history_response.json`: example response shape from `GET /api/v1/prompt-studio/optimizations/history/{optimization_id}`
- `ROLL_OUT_NOTES.md`: canary and feature-flag rollout operations for MCTS
