# Design Benchmarks

Use this directory for design-level benchmark artifacts that are referenced by design docs.

## Suggested contents

- Timestamped CSV outputs from benchmark runs.
- A short per-run notes file describing environment, dataset, and key takeaways.
- Optional plots/screenshots when they help explain regressions or improvements.

## Naming guidance

- CSV: `<topic>_<YYYYMMDD_HHMMSS>.csv`
- Notes: `<topic>_<YYYYMMDD_HHMMSS>.md`
